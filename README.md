### Projeto ADM:
1. Abra um terminal e execute o seguinte comando: 
```bash
cd desafio_inspiring/adm
npm i -g json-server
json-server --watch db.json
```
2. Sem fechar o terminal, abra outra janela e execute:
```bash
cd desafio_inspiring/adm
npm i
ng s
```
O site estará acessível em http://localhost:4200
